Hi there!

This is a set of tutorials for getting started with iOS and Android app development using the fantastic Ruby Motion.

These tutorials are for absolute beginners who know the basics of Ruby and little else. To get started, just click on the links below and follow along as I build basic apps and explore the Android and iOS APIs with you. These tutorials will be most effective if you don't just read along! I want you to be writing code, clicking links, and digging deeper as we go.

If you find that anything is amiss such as broken links, broken code, or confusing writing, please let me know! This is an open-source project so I'd love to have all the help I can get.

I don't have a solid plan for how these tutorials will ultimately be organized, so for now I'm going to dig into iOS and Android separately.
